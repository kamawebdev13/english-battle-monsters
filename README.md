
🧩 English Grammar Learning WebApp — Version 1
A web-based learning project built with native JavaScript, HTML5, and CSS, designed to make English grammar accessible, engaging, and fun for everyone.
The app introduces simple, game‑like activities guided by small, colorful monster characters that help users practice grammar concepts in an enjoyable, friendly way.
✨ What this version includes
• 	Interactive mini‑games focused on core grammar topics
• 	A playful visual style featuring unique monster guides
• 	A lightweight, fully client‑side implementation using only JS, HTML5, and CSS
• 	A foundation ready to grow into a richer learning platform
🎯 Project goal
To create an approachable and entertaining environment where learners of all ages can strengthen their English grammar skills through short, fun, and intuitive games.
